上岸QQ的鱼并不常用phpStudy，实际上phpStudy最新版本是2011B，但是我一直使用的是phpStudy 2010版。当初使用phpStudy是因为世界知名的外贸电子商务平台ZenCart太过娇生惯养，我在电脑上尝试过很多种集成环境都不能顺利安装ZenCart，唯独phpStudy 2010版才行（phpStudy 2011B也不行）。

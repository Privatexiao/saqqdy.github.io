Ambrosia这是一款完全用HTML5和CSS3写编写的WordPress主题，其结构简洁明了，并且完全是按照最新的W3C标准制作的。该主题一出就倍受网友追捧，其下载量也是相当大的。据不完全统计，该主题一个月内下载量达到4000余次。
